Python Password Generator

Overview:
Python Password Generator is a simple script that generates random passwords with a mix of alphabets, symbols, and numbers.

Getting Started:
Prerequisites:
- Python 3.x is required to run the script.

Installation:
1. Clone the repository to your local machine:
   git clone https://github.com/memoriesbytalha/passwordgenerator.git

2. Navigate to the project directory:
   cd Random Password Generator

Usage:
1. Run the script:
python password_generator.py

2. Follow the on-screen prompts to specify the number of alphabets, symbols, and numbers you want in your password.

3. The script will generate a random password and display it.

Example:
Welcome to Python Password Generator!
How many alphabets do you want to include in the password? 8
How many symbols do you want to include in the password? 4
How many numbers do you want to include in the password? 2

Your random password is: @aB3zK%8Pq
